

Подключиться к виртуалке с помощью qemu vnc server (включается параметром в virt-install или позже в конфиге виртуалки (virsh edit))

Включить Remote Desktop в Ubuntu по шагам, описанным здесь
	Note: НЕ выполнять пункт 6 (не разрешать коннект с любого интерфейса) - ниже будет понятно почему
	https://help.ubuntu.com/community/VNC/Servers

Собственно, можно коннектиться, но все бесплатные клиенты, которые я пробовал, не подключаются 
	- ругаются на неподдерживаемый тип шифрования
	Отключам шифрование: 
		gsettings set org.gnome.Vino require-encryption false
		Note: через терминал (без GUI) у меня это дело не работало. Ругалось, что нужны иксы. 
			Из терминала, запущенного из под десктопного окружения (стандартный Unity), выполнилось без проблем. 

Дальше создаем ssh туннель на порт 5900, а VNC клиент подключаем через туннель
